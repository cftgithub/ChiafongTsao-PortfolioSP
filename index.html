Rockfish Conservation

Rockfish is a vulnerable fish family that is being overfished by commercial and game fishing. This website focuses on educating users regarding Rockfish Conservation efforts and safe catch and release methods.

What's Cool...

  • JSON file to store stats of featured rockfish species to include their conservation status, habitat and image.
  • Button below each fish image allows user to get the stats from the JSON file to display in a modal.
  • Sort and display rockfish by conservation status.
  • Explanation of each conservation status is displayed on hover.
  • Rockfish identification game to reinforce the learning process.
  • Database where users can view and enter fishes caught to track effectiveness of conservation efforts.

Recette Roulette

Easy, entertaining recipe search takes the frustration out of meal preparation and turn it into an adventure inspired by the concept of roulette!

What's Cool...
