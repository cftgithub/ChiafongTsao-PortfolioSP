Rockfish Conservation

Rockfish is a vulnerable fish family that is being overfished by commercial and game fishing. This website focuses on educating users regarding Rockfish Conservation efforts and safe catch and release methods.

What's Cool...

  • JSON file used to store stats of featured rockfish species to include their conservation status, habitat and image.
  • Button click below each fish image displays stats from the JSON file in a modal.
  • Interactive buttons sorts and displays rockfish by conservation status.
  • Popovers displays explanation of each conservation status on mouse hover.
  • Includes a game designed to reinforce user's ability to identify safe species to catch. Game Demo
  • Database where users can view and enter fishes caught to track effectiveness of conservation efforts.

Recette Roulette

Easy, entertaining recipe search takes the frustration out of meal preparation and turn it into an adventure. Inspired by the concept of roulette!

What's Cool...

  • Users can determine the scope of desired search by entering keywords to narrow the search criteria or to broaden the search by leaving the search box empty and returning a complete random search.
  • Tasty API returns a recipe, ingredient list, nutritional value, image and a video instruction with every search.
  • The image of the recipe is displayed after each search, allowing users to determine whether to continue searching.
  • A modal is used to store the contents of video instruction, ingredient list, nutritional value and recipe. All can be easily accessed with the click of a button.
